module:@ynode/mongoose

This plugin adds a "mongoose" decorator to the Fastify server instance, allowing for easy access to the mongoose connection.

Parameters:
Name Type Description
fastify FastifyInstance

The Fastify instance.
options object

Plugin options, directly passed to connection.openUri.

Name Type Attributes Default Description
uri String

mongodb URI to connect to

If true, startup fails when initial MongoDB connection fails.

Optionally set a connection name. Useful for debugging
